Summary plots

Summary plots are used to find the shap value for every feature and for every sample.

  • Each dot resprensents a row of data in the dataset.

  • The color of the dot represents weather the value of the feature was high or low .

  • Its horizontal postion represents whether the feature had a positive or negative impact on the the given prediction.
