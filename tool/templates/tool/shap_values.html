SHAP Values(ie:- SHapley Additive exPlanations):

SHAP valuse break down a certain prediction in order to show how each feature contributed to the given prediction. Shap values can be used in cases where a system needs to explain it self for predicting a certain output(eg: a medical model explains what was the feature that contributed to a given prediction in order for doctor's to come up with a specfic health intervention).

How is a SHAP value calculated:

  • A baseline value is calculated for each feature contrbiuting to the prediction.

  • The Shap value then calculates how much did a particular feature drive a prediction from a baseline value to the output value.

  • The red bars shows the feature that increases the prediction and the size of the bars represents the effect of that particular feature. While the blue bars show the decreasing effect of the feature on the prediction.
