Dependence Contribution Plots

Dependence Contribution Plots are able to portray how different features interact togther and how this interaction affects the given prediction. At the same time their distribution is able to convay important information about the trends the colleration between the the two features. Dependency contribution plots are aslo easily explainable to a non-technical audience which is very useful in explaining the model to them.

  • Each dot resprensents a row of data in the dataset.

  • Its horizontal postion represents the value of the first feature of the data

  • The vertical postion of the dot represents the impact that this data row had on the prdiction.

  • The color of the dot represents the value of the secound feature that is represented by the feature.

    • Observing the slope of the pattern of the dots and their color shows the effect of the increase and decreasing the values of the feature and whether they affect each other or not.
