Premutation importants shows which features have the greatest impact on the prediction made by your model. The advantages of premutation importaance is that it is:

  • Fast to calculate.

  • Easily understood and is used alot.

  • Has the properties required to measure feature imporatnce.


    • We take the following steps to calculate preumtaion importance:

  • Train the given model.

  • We then suffel a single column of the dataset, then we compare the prediction made by the resulting dataset with the target values. Then we calculate how much the loss function suffered from the shuffling that particular feature.This performance deterioration measure the importance of that feature.

  • Return the dataset to its previous order and move to the next column and repeat the same process.
